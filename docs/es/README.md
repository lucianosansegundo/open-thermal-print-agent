# Open Thermal Print Agent

Resumen breve en espanol.

Open Thermal Print Agent es un agente local open source para imprimir tickets termicos desde aplicaciones web o POS. El MVP inicial esta enfocado en Windows, HTTP local en `127.0.0.1` e impresoras ESC/POS instaladas en el sistema.

La documentacion canonica esta en ingles en el README principal y en `docs/`.
